## reactable-forms
#### Form and schema building tools for React

### NOTICE! 
On verision 0.3 the manual form component does not currently work,
only the Auto form generation. For a working manual form, please use
version 0.2.4

### See an example
Pull down the repo

```bash
npm install
npm run example
```

### Installation
```bash
npm install reactable-forms
```

### Testing
```bash
npm test
```
